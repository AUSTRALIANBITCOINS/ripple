Overview

This class is parses chunked/streamed multipart HTTP streams. It is used by streaming MapReduce queries, and in the future streaming key-lists (once implemented on the Riak side).

def initialize(&block)

  @buffer = ""
  @block = block
  @state = :get_boundary
end
